1) Crie um repositório chamado "sistema_online"
 


2) Em todas as telas deverão possuir o seguinte menu:
 
Cursos => link para a página "cursos.html"
Turmas => link para a página "turmas.html"
Alunos => link para a página "alunos.html"
Professores => link para a página "professores.html"
 

 
3) Página index.html
 
Por enquanto terá apenas o menu



4) Página cursos.html
 
Deverá possuir um botão "Novo curso" que redirecionará para a página "formulario_curso.html".
 
Deverá possuir uma tabela com os títulos Nome e Valor.
 
Por ser tratar de um protótipo crie duas ou três linhas.  
 
No final de cada deverá possuir um "botão" de alterar e outro de excluir.
 



5) Página turmas.html

Deverá possuir um botão "Nova turma" que redirecionará para a página "formulario_turma.html".
 
Deverá possuir uma tabela com os títulos Nome do curso, data de início e data de término.
 
As demais linhas deverão ter o aspecto "zebrado".
 
Por ser tratar de um protótipo crie duas ou três linhas.  
 
No final de cada deverá possuir um "botão" de alterar e outro de excluir.
 


6) Página alunos.html
Deverá possuir um botão "Novo aluno" que redirecionará para a página "formulario_aluno.html".
 
Deverá possuir uma tabela com os títulos Nome, e-mail e telefone.
 
As demais linhas deverão ter o aspecto "zebrado".
 
Por ser tratar de um protótipo crie duas ou três linhas.  
 
No final de cada deverá possuir um "botão" de alterar e outro de excluir.
 
 
 
7) Página professores.html

Deverá possuir um botão "Novo professor" que redirecionará para a página "formulario_professor.html".
 
Deverá possuir uma tabela com os títulos Nome e telefone.
 

Por ser tratar de um protótipo crie duas ou três linhas.  
 
No final de cada deverá possuir um "botão" de alterar e outro de excluir.
 


8) Página formulario_curso.html

Deverá ter botão "Voltar" que redirecionará para cursos.html
Deverá possuir um formulário com os seguintes campos "Nome" e "Valor"                                                         
Deverá possuir botão de Ok e outro Limpar	
 


9) Página formulario_turma.html
 
Deverá ter botão "Voltar" que redirecionará para turmas.html
 
Deverá possuir um formulário com os campos:
. Curso que será um select com os seguintes valores: "Windows", "Excel", "Power BI".
. Data de início
. Data término
 
Deverá possuir botão de Ok e outro Limpar
 


10) Página formulario_aluno.html
 
Deverá ter botão "Voltar" que redirecionará para alunos.html
 
Deverá possuir um formulário com os seguintes campos: Nome, data de nascimento, endereço, e-mail e telefone
 
Deverá possuir botão de Ok e outro Limpar
 


11) Página formulario_professor.html
 
Deverá ter botão "Voltar" que redirecionará para professores.html
 
Deverá possuir um formulário com os seguintes campos: Nome, e-mail e telefone
 
Deverá possuir botão de Ok e outro Limpar



Atenção
Todas as páginas devem estar com bootstrap instalado
As tabelas devem possuir o aspecto zebrado (table-striped)
Os botões de "Novo" devem ter a cor verde.
Os botões de "Alterar" devem ter a cor amarela.
Os botões de "Excluir" devem ter a cor vermelha
Os botões de "Voltar" e "Limpar" devem ter a cor cinza

Lembre-se de colocar o menu dentro de uma tag <nav>

E o restante do código devem estar dentro de um div de classe container
